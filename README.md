# Проект на Node.JS

Очень большой проект для выбора героев для популярной MMO-RPG игры.

## Как установить?

1. Скопируйте URL репозитория
2. Вставьте его в WebStorm
3. Откройте файл package.json
4. Запустите npm run dev
